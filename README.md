# Mead Me At Midnight
This project is a tavern simulation game. The purpose of the game is to become friend, enemies, or lovers with a set amount of NPCs in the game. This game is coded with Godot.

## Installation and Run Game
1. Download the latest Godot version
2. Clone project
3. Open Godot and import cloned project
4. Once imported, click 'Run' to play the game

## Import assets from Blender
1. Export a .glb file from Blender
2. Import .glb file into the /assets folder
  - Ensure the .glb file is placed in the correct category
3. When a prompt appears, click 'Open new inherited scene'
  - This will create a new scene from your imported file
4. Make any necesssary modifications
5. Save the new scene in the /scenes folder under the correct category
